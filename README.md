## 📐 Responsive Strategy

### Mobile-First Architecture

```
MOBILE (Base)  →  TABLET (768px+)  →  DESKTOP (1024px+)
     ↓                   ↓                    ↓
 1 column            2 columns           3-4 columns
 Stack layout        Side-by-side         Full grid
 Hamburger menu     Hamburger menu       Horizontal nav
 Full-width cards   2-col cards          Multi-col cards
```

### Breakpoints

| Device | Range | Grid Columns |
|:---:|:---:|:---:|
| 📱 Mobile | 0 – 767px | 1 column |
| 📱 Tablet | 768 – 1023px | 2 columns |
| 🖥️ Desktop | 1024px+ | 3-4 columns |
| 🖥️ Large | 1200px+ | Up to 4 columns |

### Responsive Techniques

- ✅ CSS Grid with `auto-fit` / `minmax()`
- ✅ Flexbox with `flex-wrap`
- ✅ `clamp()` fluid typography
- ✅ Relative units (`rem`, `%`, `vw`)
- ✅ Mobile hamburger menu → Desktop navbar
- ✅ Touch-friendly tap targets (min 44px)
- ✅ `prefers-reduced-motion` for accessibility

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)
- Git (for cloning)
- Optional: Local HTTP server (Python, Node.js, or VS Code Live Server)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/aleemrana8/Full-Stack-Project-.git

# 2. Navigate to the project
cd Full-Stack-Project-

# 3. Open in browser (choose one method)

# Method A: Direct file
start index.html

# Method B: Python server
python -m http.server 8000

# Method C: Node.js server
npx serve .

# Method D: VS Code Live Server Extension
# Right-click index.html → "Open with Live Server"
```

### View at: `http://localhost:8000`

## ⚡ Performance

### Optimization Techniques

| Category | Implementation |
|:---|:---|
| **CSS** | Modular files, CSS variables, minimal specificity |
| **JavaScript** | Vanilla JS, Intersection Observer, passive listeners |
| **Animations** | CSS transforms (GPU), `will-change` hints |
| **Fonts** | `preconnect`, font-display swap |
| **Rendering** | No layout thrashing, fixed dimensions |
| **Bundle** | Zero dependencies, no build step needed |

### Core Web Vitals

| Metric | Target | Strategy |
|:---:|:---:|:---|
| **LCP** | < 2.5s | Minimal render-blocking, preconnect |
| **FID** | < 100ms | Lightweight JS, passive listeners |
| **CLS** | < 0.1 | Fixed dimensions, no layout shifts |
